📱 ฟีเจอร์ของระบบ

📷 สแกนสินค้าอัตโนมัติ
เมื่อคุณเปิดหน้าสแกน ระบบจะสแกนรูปภาพสินค้าของคุณโดยอัตโนมัติ และเพิ่มลงตะกร้าเลย
💰 ยอดรวมแบบเรียลไทม์
แสดงยอดรวมให้เห็นเสมอ เพื่อให้ลูกค้าเข้าใจว่าต้องจ่ายเงินเท่าไหร่
🧮 จำนวนสินค้า
สามารถเพิ่ม/ลดจำนวนสินค้าได้ทันที โดยใช้ปุ่ม + และ -
📋 ใบเสร็จชำระเงิน
หลังจากยืนยันการสแกน จะแสดงใบเสร็จที่มี QR Code ชำระเงิน และสามารถถ่ายรูปสลิปได้
📱 QR Code ชำระเงิน
ใส่ชื่อบัญชีธนาคารของคุณ จะสร้าง QR Code อัตโนมัติ และแสดงในใบเสร็จ
📸 ถ่ายรูปสลิปโอนเงิน
เมื่อลูกค้าโอนเงิน สามารถถ่ายรูปสลิปมาใส่ในระบบได้เพื่อเก็บหลักฐาน
⚙️ จัดการสินค้า
ในหน้าจัดการ สามารถเพิ่ม ลบ แก้ไขสินค้า และอัพโหลดรูปสินค้าได้
🔒 ระบบ PIN
เพื่อความปลอดภัย ต้องใส่รหัส PIN 6 หลักก่อนใช้ระบบ (PIN ปัจจุบัน: 123456)
💾 เก็บข้อมูลในอุปกรณ์
สินค้า QR Code สลิป และรายการทั้งหมดเก็บอยู่ในอุปกรณ์ของคุณอย่างปลอดภัย
